My Projects

Here is a list of most of the projects that I've worked on and what I've learned and used with them. The first 3 from the main page might also be here, if they are, they will have some more additional information about them. Most of these projects should have a link to the code/project, but some will not have any code at the moment, I have yet to upload some.

Make A Website for Change

As I stated on the main page, this was a project that had the theme of, "If you wanted to bring awareness for something or to create change in your community, what website would you build to spread the word?". I chose Open-Source software as my cause because I believe Open Source Software can be very beneficial to the average user. Open source software is important to me because it gives users the freedom to do the things they want while also respecting their privacy--unlike proprietary software. The goal of this website was to get the average user to consider using Open Source Software, as it leads to more benefits when compared alongside of proprietary software.

Tetris

This was a project making the popular game, Tetris in C++ with a library called Raylib. It was a very minimal videogame library/framework, only having the documentation inside of a single header file. This got me to work with C++ again, it had been a while since I had done anything with C++. This was a massive refresher. I got introduced to new concepts within C++ that I either barely knew or didn't know at all.

MilkOS, A BareBones OS written in Assembly and C!

This was a really fun project getting to test myself with bascially having to write everything, as there was no standard library that I could invoke. Unfortunately, the code has been lost to time. I was only able to find one image of it working at all, but it was very barebones. Theoretically, I could get it working on bare metal, but never tried.
